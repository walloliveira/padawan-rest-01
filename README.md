# Criar CRUD de usuário

Eu como Lider técnico quero cadastrar/editar/excluir o usuário.
Para cadastrar vou informar os dados abaixo:
- Nome
- Sobrenome
- Data de nascimento
- Nome do usuário
- O que ele é?
  - Administrador
  - Supervisor
- C.P.F.
- E-mail

## Critérios de aceitação
- O CPF é obrigatório
- O nome do usuário é obrigatório
- Nome e sobrenome são obrigatórios
- Data de nascimento é obrigatória
- E-mail é obrigatório
- O usuário deve ser maior que 18 anos
- O Nome do usuário é único
- O CPF deve ser válido
- O CPF não pode ser alterado na edição
- O CPF deve ser único
- E-mail deve ser único
- A Data de nascimento não pode ser alterada na edição
- O Nome do usuário não pode ser alterado na edição
- O que ele é não deve ser apresentado na recuperação dos usuários
